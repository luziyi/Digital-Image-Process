# 数字图像处理课程

### 课程介绍

主要学习数字图像获取，颜色空间，图像着色，图像边缘检测，图像插值，不变特征提取，

### 实验内容

实验一：边缘检测

实验二：图像插值

实验三：图像低光增强

实验四：目标检测
